# 1. 新建项目

官方文档

# 搭建博客

# 安装本地依赖
yarn -D add vuepress

# 新建文件夹，日后的博客文档均置于此
mkdir docs

# 新建一个 md 文件
echo 'hello vuepress' > docs/README.md

# 启动服务
npx vuepress dev docs

启动之后，打开提示的链接，应该能看到这种情况：

博客就搭建成功了。

在 docs/README.md 随便修改两下，浏览器会热更新：

<!-- docs/README.md -->
你好哇，vuepress！

博客会自动更新：

# 打包部署

# 打包项目
npx vuepress build docs

之后，会在 docs 下生成一个 dist 目录：

docs
└── .vuepress (可选的)
    └── dist 
        ├── index.html
        ├── 404.html
        └── assets
            ├── index.html
            ├── index.html
            └── 404.html

这个目录即可用来进行项目部署。

vuepress 这个项目是用来支撑 Vue 及其子项目的文档需求，是一个静态网页。build 之后就不再变化了，可以常规部署，也可以在 gitpages 里部署。
