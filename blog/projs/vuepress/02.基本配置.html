# 配置文件 config.js

为了给博客网站添加更多自定义内容，需要添加一个配置文件，其位置如下：

docs
└── .vuepress (可选的)
    └── config.js

其内容应当导出一个对象：

// docs/.vuepress/config.js
module.exports = {
    title: '春虫', // 网站标题
}

重启服务后，可见更新：

注意：配置文件变更后，无法自动更新，必须重启一下服务。
