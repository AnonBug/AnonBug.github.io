# 安装

# 1. 安装 licence

1.1 ArcGIS_License_Manager_Windows_106_161708.exe文件双击打开，此时是解压，建议放在非C盘，安装后可删；

1.2 解压后，正式安装，无脑下一步即可。

1.3 停止服务

在开始菜单-->ArcGIS-->License Manager-->License Server Administrator

# 2. 安装 desktop

2.1 双击打开，同样是解压，安装后可删。

2.2 无脑下一步即可。

2.3 安装完成

# 破解

# 3. 补丁文件替换

3.1 将 license 下的补丁文件拷进 license 相应安装bin文件夹。

3.2 将 desktop 下的补丁文件拷进 arcgis 的相应 bin 文件夹。

# 4. license 许可重新读取

4.1 开始服务

4.2 重新读取许可

# 5. desktop 设置

5.1 选择 advance 浮动版

5.2 许可管理器输入 localhost

5.3 破解完成，可点击可用性查看破解状态。

# 6. 中文语言包安装

6.1 双击安装，无脑下一步

6.2 在 desktop 的设置页面选择高级，切换语言（根据经验，安装好语言包后，会自动切换成中文语言，不过还是确认一下为好）

# 爬坑

# 执行ArcGIS Administrator时，报错：无法启动该程序，因为计算机中丢失api-ms-win-crt-conio-|1-1-0.dll”

解决方案来源： https://www.fangfaku.net/e487392/

  1. 网上检索下载缺失 dll 文件

https://www.jb51.net/dll/api-ms-win-crt-conio-l1-1-0.dll.html#downintro2

  1. 将所下载 dll 文件存到 C/Windows/System32/下(win7系统)

  2. 制作 bat 文件，文件名自定，位置随意，代码如下，务必原样复制

@echo 开始注册
copy api-ms-win-crt-conio-l1-1-0.dll %windir%\system32\
regsvr32 %windir%\system32\api-ms-win-crt-conio-l1-1-0.dll /s
@echo api-ms-win-crt-conio-l1-1-0.dll注册成功
@pause

  1. 双击执行 bat 文件，当出现如下结果，证明添加成功

  2. 重启电脑！！！
